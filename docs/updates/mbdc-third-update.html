Tags

So, you might've seen these "#'s", and i'm calling those tags. They are considered a lot of different things to a lot of people. Hash tags, topics, pound signs. I'm calling them tags, because it's short and simple. Eventually I'm going to have hover and dropdown tags in the navigation bar for each "tag / topic".

A tag is just a collection of a topic. So, for the image above, you'll see #currently. From there you can see the collection that falls under #currently: Currently Listening, Reading, and Playing. So that's what you'll see all over the site now. I have tags all over the review section.

That is mostly to keep things tidy, but for future proofing as well. As I continue to improve the site, things like tags are needed.
