A Game of Thrones Review

April 3, 2023

in Progress

Book Reviews

Written Review

First off, I've seen the entire Game of Thrones show. I don't remember much of the show, other than.. well I'll add that to the Spoilers Section.. But I don't remember much of the show and I'm excited to read this series, and I'm just starting with Book One right now as I work through my Too Be Read list. I'm excited to see what this book does!

Spoilers

Just a bit of preamble here.. the way this review will be is a lot of different from others. I'm wrote notes on every single chapter that I read. Which is completely spoiler, so this is where the spoilers start.. read at your own risk!

Notes As I Read

Prologue

Oh whoa! This was crazy! This was terrifying.. wow. I'm really amazed by this so far. I like the writing so far, really. I'm intrigued so far!

Bran

Okay, here we go! This is so good. I really like this story so far. I like the way that these characters are completely different - I really like this. I'm interested in this story, being that I'm in the middle of some other series like Wheel of Time and The Stormlight Archive, this is a good book so far.

Catlyn

Wow, this is big. I really like this so far.

Daenerys

Okay now I'm seeing a big difference between what I've read so far. This book has been pretty brutal so far, and scary (with this Other) and I've read some violence before, but this - and I mean, I've seen the show, so I know it's a bit graphic with sex and other things, but hmm I don't know. It'll be different to read, I think. I'm very worried for Dany here. I know what happens, since I've seen the show, and the remembering of the show is coming back, so I'm very worried about Dany.

Jon

Wow, I love Jon so much haha. He’s great! And Tyrion was very nice to Jon. I like that.

I’m noticing that all these characters, save a few, are young characters. I mean really young. Early teens to even younger than 10 and it is portrayed very well.

Catelyn

Okay, our first foray into a bit of sexual content here. This is, for me, my first reading of any sexual content - I did start reading Fifty Shades of Grey, but I haven't gotten very far into that book so I haven't seen much of that, but this is very mild so far. Her loins, his seed, that's what we got here.

I'm really enjoying this book, I really am, the treachery, the scandals. This is good!

Arya

I love Arya - I'm really happy to see her and Jon. She's a great character so far!

I am still enjoying this book, but I want to get back to Wheel of Time quickly. I might start with the Audiobook and read the book at the same time... let's see! Turns out I canceled my Audible subscription so I have no more credits left!

Bran

The infamous scene, at least I know this scene from the show.. but here it is on paper. Very interest from Bran's perspective - 'he was hurting her, but she wasn't pushing him away'.. interesting.
